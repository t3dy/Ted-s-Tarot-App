About This Project

Arcana Encyclopaedia

A digital humanities project presenting scholarly research on the history, terminology, and interpretive traditions of the tarot. Built using the DH Framework architecture (SQLite → Python → static HTML → GitHub Pages).

The underlying database contains 78 cards with 225 per-tradition meanings across the Rider-Waite-Smith, Thoth/Crowley-Harris, and Marseille traditions. Every card carries Golden Dawn correspondences: Hebrew letters, Tree of Life paths, Sephirotic attributions, and astrological assignments.

Interpretive Philosophy

This project approaches tarot as a psychological and narrative tool, not a fortune-telling device. It is grounded in evidence-based historiography (the Dummett school), multi-tradition fluency (RWS, Thoth, Marseille), and structural reading methods (suit prevalence, trump density, number sequences).
